# No Way Out

Jogo 2D top-down de polícia e ladrão desenvolvido em Unity.

## Sobre o jogo
Em *No Way Out*, o jogador controla um fugitivo que precisa coletar moedas e sobreviver o máximo de tempo possível enquanto é perseguido pela polícia. Conforme o jogo avança, os inimigos ficam mais rápidos e novas ameaças aparecem, incluindo um policial atirador.

## Como jogar
- Mover: WASD / Setas / Controle Xbox
- Objetivo: coletar moedas, aumentar o score e sobreviver o máximo de tempo possível
- Cada moeda coletada:
  - aumenta o score
  - adiciona tempo ao cronômetro
- O jogo termina quando:
  - o tempo acaba
  - ou o jogador perde todas as vidas

## Mecânicas implementadas
- cronômetro
- score
- vidas
- inimigos perseguidores
- inimigo atirador
- progressão de dificuldade por ondas
- suporte a joystick/controle Xbox
- áudio de fundo e efeitos sonoros
